# Tenant Usage

## Prerequisites

- Role is `tenant_admin`.
- Billing summary data exists for selected period.

## Steps

1. Open `Tenant > Usage`.
2. Input billing period (`YYYY-MM`).
3. Click **Load Usage**.

## Expected Result

- Usage summary table is shown.
- Usage distribution chart shows billable and overage percentages.

## Troubleshooting

- Empty result: verify data for billing period.
- `404` summary not found: recalculate summary from admin flow.
