# Installation Manual

This manual explains how to run the project locally and verify the core services.

## What You Will Set Up

- `backend/` Go services:
  - gateway (`:8080`)
  - control plane (`:8081`)
- `frontend/` Vite React console
- `docs/nextra/` documentation app

## Steps

1. Complete [Prerequisites](./prerequisites).
2. Follow [Local Setup](./local-setup).
3. Optionally review [Basic Deployment](./deploy-basic).
4. If something fails, use [Troubleshooting](./troubleshooting).

## Expected Result

- Gateway health endpoints respond.
- Control plane admin APIs are reachable.
- Frontend loads and can access Public/Admin/Tenant modules.
